VAMPS Documentation
===================

VAMPS (VAriably saturated soil Model with Plant and canopy System) is a
one-dimensional model for the flow of water through a forested or agricultural
ecosystem.  The core is a Richards-equation solver for unsaturated flow through
the soil.  Since version 1.0 a Python package (``vampspy``) is provided for
driving VAMPS entirely from Python.
