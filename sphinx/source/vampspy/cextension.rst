_vampscore C Extension
======================

``vampspy._vampscore`` is the compiled C extension that wraps the VAMPS soil
model.  It exposes a low-level Python API used internally by
:class:`~vampspy.bmi.VampsBmi` and by the high-level :func:`~vampspy.run`
convenience function.
